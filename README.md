# \'Lo Patio Website

## Project setup
```
yarn install && pipenv install
```

### Compiles and hot-reloads for frontend development

```
yarn serve
```

### Serves backend development server

```
pipenv shell && python manage.py runserver
```

### Compiles and minifies for production
```
yarn build
```

yarn lint
